# BookManagementApp
An Android app for showing books, creating books, updating books and deleting books.

## Prerequisite
+ Android 12  

Tested on: 
  + Pixel 6(Android13, virtual)
  + Pixel 6(Android13, physical)
  + Pixel 5(Android14, physical)
